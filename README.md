# Movie-Ticket-booking
🎬 Movie Ticket Booking System

A beginner-friendly Python project that simulates a movie ticket booking system using functions, loops, lists, and dictionaries.

📌 Features

- Display available movies with ticket prices
- Book movie tickets
- Select multiple seats
- Generate a random Booking ID
- Cancel booked tickets
- Restore cancelled seats
- View booking history
- Validate movie names and seat selection
- Menu-driven interface

🛠️ Python Concepts Used

- Functions
- Dictionaries
- Lists
- Loops ("for", "while")
- Conditional Statements ("if", "elif", "else")
- User Input Handling
- String Methods
- Random Module
- Input Validation

📂 Project Workflow

1. Display available movies.
2. Select a movie.
3. Enter number of tickets.
4. Choose available seats.
5. Calculate total ticket price.
6. Generate Booking ID.
7. Save booking history.
8. Cancel tickets if required.
9. Display complete booking history.

🚀 Future Improvements

- Save booking history using file handling.
- Prevent duplicate Booking IDs.
- Add user login system.
- Add payment simulation.
- Use Object-Oriented Programming (OOP).
- Improve seat layout visualization.
